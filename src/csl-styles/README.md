# Bundled Citation Styles

This directory contains the 10 most popular CSL (Citation Style Language) citation styles, bundled with Lighthouse for convenience.

## Included Styles

1. **APA** (apa.csl) - American Psychological Association, 7th edition
   - Used by: Psychology, Education, Social Sciences

2. **Chicago** (chicago-author-date.csl) - Chicago Manual of Style (author-date)
   - Used by: History, Arts, Humanities

3. **MLA** (modern-language-association.csl) - Modern Language Association, 9th edition
   - Used by: Literature, Languages, Humanities

4. **Harvard** (harvard-cite-them-right.csl) - Harvard referencing style
   - Used by: UK universities, Business

5. **IEEE** (ieee.csl) - Institute of Electrical and Electronics Engineers
   - Used by: Engineering, Computer Science

6. **Vancouver** (vancouver.csl) - Vancouver citation style
   - Used by: Medicine, Biomedical Sciences

7. **AMA** (american-medical-association.csl) - American Medical Association
   - Used by: Medicine, Health Sciences

8. **Nature** (nature.csl) - Nature journal style
   - Used by: Natural Sciences

9. **Elsevier Harvard** (elsevier-harvard.csl) - Elsevier's Harvard style
   - Used by: Multi-disciplinary publishing

10. **ACM** (association-for-computing-machinery.csl) - Association for Computing Machinery
    - Used by: Computer Science

## Source

All styles are from the official [Citation Style Language](https://github.com/citation-style-language/styles) repository and are licensed under Creative Commons Attribution-ShareAlike 3.0 Unported (CC BY-SA 3.0).

## Downloading Additional Styles

Use the "Download more..." button in the Project settings to search and download any of the 10,000+ citation styles available from the CSL repository.

Downloaded styles are saved to `{plugin-dir}/csl-styles/downloaded/`.
