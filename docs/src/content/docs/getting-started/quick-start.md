# Quick Start Guide

This guide walks you through creating your first Lighthouse project and understanding the core workflow.

## Step 1: Create Your First Project

1. Open the **Command Palette** (Cmd/Ctrl+P)
2. Run **Lighthouse: Create new project**
3. Enter a name for your project (e.g., "My Novel")
4. Select a root folder for your project (or create a new one)
5. Click **Create project**

Your project is now created and set as active.

## Step 2: Enter the Writing Workspace

Click the compass icon in the left ribbon, or run **Lighthouse: Open writing workspace** from the Command Palette.

This opens Lighthouse's two-panel layout:

- **Library** (left sidebar) — browse your project as Groups and Sheets
- **Inspector** (right sidebar) — project stats, writing activity, and the current file's outline

Obsidian's native file explorer steps aside for the duration, so the left sidebar is exclusively Lighthouse's. Toggle back at any time with the same ribbon icon, or run **Lighthouse: Exit writing workspace**.

## Step 3: Organize with Groups

Every project starts with an **Extras** group — a place for research, character notes, and anything else that shouldn't count toward your word goal. Add as many top-level Groups as you like for your actual writing (chapters, sections, drafts):

1. Click the **+** icon in the Library header (or right-click in the Library and choose **New group**)
2. Name the group and pick an icon
3. Click **Done**

Groups can be nested — right-click any group for **New group**, **New note**, **Edit group…**, and **Delete**. Selecting a group shows its notes ("Sheets") in the column beside it.

## Step 4: Set a Goal (Optional)

1. Open the project's "⋯" menu in the Library header and choose **Edit project…**
2. Set a **Word count goal**
3. Optionally set a **Deadline** and **Daily writing goal** for pacing
4. Save

The Inspector's **Overview** tab will now show a goal ring and, once you have writing history, a 13-week activity heatmap.

## Step 5: Start Writing!

Open any note in your project and start writing. Lighthouse will:

- Track your word count in real-time, at the file, group, and project level
- Update the status bar count automatically
- Track your daily and session progress
- Show progress toward your goal in the Inspector

## Understanding the Inspector

The Inspector (right sidebar) has three tabs:

- **Overview** — goal ring, project stats, and the 13-week writing activity heatmap
- **Stats** — live word counts, deadline pacing, read/speak time estimates, and your writing streak
- **Outline** — headings in the active file; click to jump

## Tips for Success

### Let Extras do its job
Keep research, character notes, and planning material in the Extras group — it's automatically excluded from your word count, so your goal always reflects actual writing.

### Use Multiple Projects
Don't try to fit everything into one project. Create separate projects for:
- Different books or manuscripts
- Blog series vs standalone posts
- Academic papers vs creative writing

### Check Your Progress Daily
The "Today" counter in the Inspector's Stats tab resets at midnight, giving you a clear view of daily productivity.

### Don't Obsess Over Numbers
Word counts are useful, but they're just one metric. Focus on quality writing, not just quantity.
